# Assets móviles

Los binarios no se incluyen en el repositorio para evitar errores al crear el pull request.

Cuando vayas a generar builds reales, coloca aquí estos archivos PNG:

- `icon.png` — icono principal de la app.
- `adaptive-icon.png` — foreground del adaptive icon Android.
- `splash.png` — imagen de splash.

Después puedes reactivar las referencias en `mobile/app.json` si quieres usar assets personalizados.
